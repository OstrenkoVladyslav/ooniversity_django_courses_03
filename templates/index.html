PyBursa

Description

The course starts by overviewing Python, and then goes on to study Python's collections - Lists, Tuples and Dictionaries etc. We introduce the development environments and other resources available, and take a look with practical exercises at test procedures and good code design.

Program

Python language fundamentals

Introduction to Python; Python - Fundamentals; Conditionals and Loops; Lists and Tuples Functions, Modules and Packages.

Python, Object Orientation, Object use

Python - using functions, objects and modules; Object Oriented Python; Dictionaries Exceptions; File Handling.

Python uses and applications

Already written modules; Python on the Web, Django framework; Python and SQL databases.
